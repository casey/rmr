This is a visualization of the activity of the Bitcoin network.

Bitcoin is peer-to-peer currency.

It is backed by a global network of computers all speaking the same open-source protocol.

To move funds from one user to another, the sender creates a transaction and broadcasts it across the network.

Some computers, known as "Bitcoin miners", gather these transactions into blocks.

New bitcoin are rewarded to the miner with every block mined. The network adjusts the difficulty of mining so that new blocks are mined roughly every ten minutes.

Each new block contains a pointer to a previous block, and so forms the head of a chain of blocks pointing back to the first block ever mined.

The chain of blocks serves as a secure distributed ledger. Anyone can examine it to check how many bitcoin they have, as well as to verify that new transactions are valid.
